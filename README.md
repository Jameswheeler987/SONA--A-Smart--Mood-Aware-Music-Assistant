# Sona – Mood-Based Music Recommendation System

Sona is a web-based application that provides music recommendations based on user-selected mood and activity. The system integrates with the Spotify Web API to retrieve real-time song data and present it through a simple and intuitive interface.

## Features
- Mood and activity-based recommendations
- Integration with Spotify Web API
- Dynamic content retrieval
- Randomised results for variation
- Display of album artwork and track details

## Technologies Used
- Python (Flask)
- HTML & CSS
- Spotify Web API

## How to Run
1. Clone the repository
2. Install required dependencies (e.g. Flask, requests)
3. Add your Spotify API credentials
4. Run the Flask application
5. Open in browser

## Project Purpose
This project was developed as part of a final year Computer Science degree to demonstrate the design and implementation of a context-aware recommendation system.
